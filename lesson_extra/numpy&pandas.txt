- Инструменты для работы с данными
Нужны для их анализа и проведения первичной аналитики
- pandas - https://pandas.pydata.org - взаимодействие с табличными данными. Одна из самых известных и широко используемых библиотек. Такой аналог excel для python =) Позволяет группировать, фильтровать, обрабатывать данные имеющие табличную структуру
- matplotlib - https://matplotlib.org, seaborn - https://seaborn.pydata.org - библиотеки для визуализации данных в python. matplolib - базовый и более простой, seaborn - более продвинутые и сложные визуализации
- statsmodels - https://www.statsmodels.org/stable/index.html - библиотека, реализующая множество статистических инструментов в Python
- numpy - https://numpy.org - низкоуровневый модуль, реализующий алгоритмы и структуры данных для научных вычислений. Сам по себе именно для журналиста может быть не очень востребован, но он лежит в основе все библиотек из этого и следующего раздела, и нет-нет, да всплывает наверх. Поэтому познакомиться с ним стоит.
